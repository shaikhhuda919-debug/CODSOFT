## Overview
Quiz Master is a PHP & MySQL based online quiz application with user authentication, quiz playing,
result evaluation, and certificate generation.

## Features
- User Registration & Login
- Secure Session-based Authentication
- Quiz Play System
- Result Page with Score
- Certificate Generation (Printable / Save as PDF)
- Admin Panel (optional)
- Responsive UI with Bootstrap

## Installation
1. Install XAMPP
2. Copy project to htdocs
3. Import database.sql
4. Run http://localhost/Quizmaster.Codesoft/

## Certificate
Certificate shows:
- User Name
- Email
- Score
- Date & Time
- Motivation Message
